# Naming Shortlist (Non-final)

## Constraints

- Must not use “AgentsKB”, “AgentKB”, or confusingly similar spellings.
- Must communicate: grounded, verified, developer knowledge base; self-expanding; private control.
- Must be usable as:
  - product name
  - Python package name
  - API service name
  - MCP server name

- **GroundedKB**
- **VerifiedKB**
- **SourceLock KB**
- **CiteOnly KB**
- **SpecKB**
- **DocProof KB**
- **AnchorKB**
- **TraceKB**
- **ProofStack KB**
- **AtlasKB**
- **ForgeKB**
- **KeystoneKB**
- **ClearSource KB**
- **AuthorityKB**
- **CanonKB**
- **AuditKB**
- **VerityKB**
- **VouchKB**
- **CiteChain KB**
- **TrustKB**

## Selection checklist

- Trademark/confusability check vs “AgentsKB”
- Package-name availability (npm + PyPI)
- Domain availability
- Pronunciation / memorability
